# `get-specs`

Scripts to gather computer specifications.

## How to Run The `get-specs` Scripts

### Linux

```bash
curl -fsSL https://raw.githubusercontent.com/raykrajci/miscellanea/refs/heads/main/get-specs/linux.sh | bash
```

### MacOS

```bash
curl -fsSL https://raw.githubusercontent.com/raykrajci/miscellanea/refs/heads/main/get-specs/macos.sh | bash
```

### Windows

```powershell
irm https://raw.githubusercontent.com/raykrajci/miscellanea/refs/heads/main/get-specs/windows.ps1 | iex
```
